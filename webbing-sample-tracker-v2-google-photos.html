Connect Google Sheets

  1. Create a Google Sheet with 2 tabs: Samples and Webbing Colors
  2. Go to Extensions → Apps Script
  3. Delete any existing code, paste the script below, then click Save
  4. Click Deploy → New deployment, select type Web app
  5. Set "Who has access" to Anyone, click Deploy, authorize when prompted
  6. Copy the Web App URL it gives you and paste it below

This syncs Samples, Webbing Colors, and Webbing Groups. Photos upload to Google Drive and the Google Sheet stores the PhotoURL link.
